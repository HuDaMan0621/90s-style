Notes About Python

  • Comments start with #.
  • Python supports negative indexing. [-1] will return the last character in a string.
  • The in keyword is an operator that tests for membership.
  • ints, floats, strings, tuples are immutable.
  • lists are mutable and dynamic.
  • String methods return a new string object.
  • List methods modify the target list in place.
  • A "singleton list" is a list with only one item in it.

Python Tools, Functions, and Methods

  • type() will tell you the data type of any value you pass in to it.
  • input() prompts the user for input and stores their input in a variable as a string value.
  • ord() will give you the ordinal (ASCII) value of any character value you pass in to it.
  • chr() will give you the character value of any ordinal value you pass in to it.
  • range() accepts a number as its argument and returns a non-inclusive sequence of numbers to loop over.
  • .lower() is a string method that does not mutate the original string; it returns a lowercase copy of the original string.
  • .upper() is a string method that does not mutate the original string; it returns an uppercase copy of the original string.
  • .append() is a list method that mutates the list it is called on by adding a single item to the end of the list.
  • .extend() is a list method that mutates the list it is called on by concatenating the arguments passed in to it to the list.
  • slice() is a funciton that returns values at the specified index(es). slice() takes 3 agruments: start position, end position, and increment (step) value.
  • .pop() is a list method that mutates the list it is called on and returns the item at the end of the list.
  • If you want to remove an item from a list, but don't want the return value of that item, use the del keyword: del list[index]
  • min() is a function that returns the smallest of input values. Input value can be a string, list, tuple, etc.
  • max() is a function that returns the largest of input values. Input value can be a string, list, tuple, etc.
  • .clear() is a list method that removes all items from the list it is called on.
  • .sort() is a list method that sorts all items in ascending order in the list it is called on.
  • .reverse() is a list method that reverses the order of all items in the list it is called on.
  • .copy() is a list method that returns a shallow copy of the list it is called on.
  • [:] safely copies a list/string.

Tuples and Lists

Use Tuples instead of Lists when:

  • Data shouldn't be modified.
  • A Python Dictionary requires keys that are of an immutable type.
  • FYI: Program execution is faster when manipulating Tuples versus Lists.

Python REPL

Stands for Read, Evaluate, Print, Loop. REPLs exist for many languages.

Prompts and runs code over and over.

To run scripts in the REPL, run import filename(your_file.py).

"Falsy" values in Python

  • The number 0.
  • None type.
  • "" Empty String.
  • [] Empty List.
  • {} Empty Set.
  • All other values are "Truthy".

Python Errors

Use try and except keywords to handle errors in Python and ensure good user input.

If code can cause an error, put it in a try body. Use except to handle specific errors.

Traceback lets you know you've got an error somewhere in your code.

NameError is an error that is the result of an issue with variable labels.

TypeError is an error that is the result of an issue with variable values.

HTML Elements

Anatomy of an HTML Element:

  • HTML is the structure of all websites.
  • There are roughly 80 tags recognized by all modern browsers.
  • HTML tags tell the browser what kind of object to create.
  • HTML elements are meant to be nested in order to create a hierarchical structure.
  • HTML elements can have attributes that are defined in the opening tag.

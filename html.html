HTML Elements

Anatomy of an HTML Element:

  • HTML is the structure of all websites.
  • There are roughly 80 tags recognized by all modern browsers.
  • HTML tags tell the browser what kind of object to create.
  • HTML elements are meant to be nested in order to create a hierarchical structure.
  • HTML elements can have attributes that are defined in the opening tag.

    • Some elements require an attribute:

    • a tags require an href attribute that points to another website.
    • img tags require a src attribute that points to the location of the image. img is a self closing tag!

Useful Tags

  • div is a container that can hold text or other elements.
  • p is used to structure text into paragraphs.
  • ul is used to structure text into a bulleted, unordered list.
  • ol is used to structure text into a numbered, ordered list.
  • li is used to structure individual items in a ordered and/or unordered list.
  • br is a self-closing tag used to create a line break (carriage return).
  • hr (horizontal rule) is a self-closing tag used to create a horizontal line between elements.

Keyboard Shortcuts
ctrl a Jumps to the beginning of prompt line.
ctrl e Jumps to the end of prompt line.
ctrl c Clears prompt and gives you a fresh line.
ctrl cmd space Emoji!
ctrl r When used in bash, allows you to search through previous commands.
cmd shift h Opens the Home folder of current macOS user account.
ctrl l Clears the screen of the terminal.
option up or option down Moves a line up or down in VS Code.
Shift tab De-dent highighted text
cmd / Comment/uncomment any selected text in VS Code.
cmd option down This will select lines below your cursor in VS Code.

Escape Sequences
\n Escape sequence for newline.
\t Escape sequence for tab.

CSS Styles

Anatomy of a CSS Style:

  • CSS provides the style of all websites.
  • There are roughly 259 different style properties.
  • CSS Styles can be created in 3 different ways:
    1. Inline
    2. Using the style tag
    3. Linking to a .css file
  • Not all CSS Styles behave the same way!
  • CSS Styles operate in box-shape unless you explicitly shape the box differently.
